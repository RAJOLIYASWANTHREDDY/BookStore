# CodeBook

A React-based online bookstore application for browsing and purchasing eBooks.

## Features

- Browse eBooks collection
- Search and filter products
- User authentication (Login/Register)
- Shopping cart functionality
- Order management
- Dark mode support

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `npm run server`

Starts the JSON server with authentication on port 8000.

### `npm test`

Launches the test runner in the interactive watch mode.

### `npm run build`

Builds the app for production to the `build` folder.

## Technologies Used

- React
- React Router
- Tailwind CSS
- JSON Server Auth
- React Toastify
